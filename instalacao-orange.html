Manual de instalação

Instalando o Orange Data Mining

Passo a passo para baixar, instalar e abrir sua primeira base de dados — no Windows e no macOS. Tudo é gratuito e não exige conhecimento de programação.

Ir para a página de download Ver o passo a passo
Passo 1 · onde baixar

Baixe o instalador no site oficial

Acesse orangedatamining.com/download. O site detecta seu sistema automaticamente, mas você também pode escolher manualmente entre Windows e macOS.

↑ A página de download. Use o botão correspondente ao seu sistema.

Windows

Windows 10 ou 11 · 64 bits
  1. Clique em “Download for Windows”.Começa a baixar um arquivo Orange3-x.x.x-...exe (instalador).
  2. Abra o arquivo baixado (pasta Downloads, dê duplo clique).Se o Windows mostrar um aviso de segurança, clique em “Mais informações” → “Executar assim mesmo”.
  3. Avance no instalador clicando em Next e aceite os termos.Pode manter todas as opções padrão.
  4. Clique em Install e aguarde a barra concluir.A instalação leva alguns minutos.
  5. Clique em Finish.Um atalho Orange aparece no Menu Iniciar e na Área de Trabalho.
Alternativa · versão portable

Não tem permissão de administrador (computador da empresa/laboratório) ou prefere rodar de um pen drive? Use a versão portable, que não exige instalação.

  1. Na página de download, escolha a opção “Portable orange” (ou Standalone / Portable) em vez do instalador.Baixa um arquivo .zip.
  2. Extraia o .zip para uma pasta de sua escolha (ex.: Área de Trabalho ou um pen drive).Botão direito no arquivo → Extrair tudo.
  3. Abra a pasta extraída e dê duplo clique em Orange.exe.O Orange abre direto, sem instalar nada no sistema.
Instalador: ~250 MB. Não precisa instalar Python à parte — já vem incluído. A versão portable não cria atalhos e não escreve no registro do Windows.

macOS

Intel e Apple Silicon (M1/M2/M3)
  1. Clique em “Download for macOS”.Baixa um arquivo Orange3-x.x.x.dmg.
  2. Dê duplo clique no .dmg baixado.Abre uma janela com o ícone do Orange e a pasta Applications.
  3. Arraste o ícone do Orange para a pasta Applications.É assim que o app é instalado no Mac.
  4. Abra o Launchpad (ou pasta Aplicativos) e clique em Orange.
  5. Se aparecer “não é possível abrir, desenvolvedor não identificado”:Clique com o botão direito no ícone → Abrir → Abrir. Ou: Ajustes do Sistema → Privacidade e Segurança → “Abrir Assim Mesmo”.
Tamanho: ~250 MB. Só é preciso liberar a abertura uma vez.
Passo 2 · rodando pela primeira vez

Abrindo o Orange

Ao abrir, o Orange mostra uma tela de boas-vindas. Feche-a (ou clique em New) para chegar ao Canvas — a área em branco onde você monta o fluxo. À esquerda fica a caixa de widgets.

↑ O Canvas com um fluxo simples: File → Data Table. Para adicionar um widget, arraste-o da caixa à esquerda ou dê duplo clique no Canvas e procure pelo nome.

Para começar do zero

  • Arraste o widget File (categoria Data) para o Canvas em branco.
  • Para abrir um workflow pronto (.ows): menu File → Open e selecione o arquivo.
  • Ligue dois widgets clicando na borda de um e arrastando a linha até o outro.
Passo 3 · carregando dados

Carregando um arquivo com o widget File

O widget File é a porta de entrada dos dados. É por ele que você abre os arquivos .csv ou .xlsx usados nas práticas.

3.1

Abra o widget

Dê duplo clique no widget File que está no Canvas. Abre a janela de configuração mostrada ao lado.

3.2

Selecione o arquivo

Com a opção “File” marcada, clique no botão “…” (ou na pasta) e localize o arquivo no seu computador. Selecione o .csv baixado e clique em Abrir.

3.3

Confira que os dados entraram

Logo abaixo aparece um resumo: número de instâncias (linhas) e atributos (colunas). Para ver a tabela, ligue o File a um Data Table e dê duplo clique nele.

Problemas comuns

Se algo der errado

? No Mac aparece “app de desenvolvedor não identificado”.

Clique com o botão direito no ícone do Orange → Abrir → confirme em Abrir. Ou vá em Ajustes do Sistema → Privacidade e Segurança e clique em “Abrir Assim Mesmo”. Só precisa fazer isso na primeira vez.

? No Windows o SmartScreen bloqueou o instalador.

Clique em “Mais informações” e depois em “Executar assim mesmo”. O instalador do Orange é seguro — o aviso aparece por ser um programa baixado da internet.

? O widget File mostra erro de caminho ao abrir um .ows.

Isso acontece quando o arquivo foi montado em outro computador. Dê duplo clique no File, clique em “…” e aponte novamente para o .csv na sua pasta Dados/.

? Os números aparecem como texto / acentos quebrados.

No widget File, confira o separador e a codificação. Para nossos arquivos use separador vírgula e codificação UTF-8. Arquivos .xlsx evitam esse problema.
