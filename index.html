Workflows do Orange

.ows · pipelines prontos
  • Dia 1 · Data Mining feito em aula Pipeline montado ao vivo: File → Edit Domain → Impute → Select Rows → Preprocess, com Box Plot, Distributions, Scatter e Column Statistics .ows
  • Dia 1 · Data Mining (referência) Exploração: File → Data Table → Distributions, Scatter, Preprocess, Sampler .ows
  • Dia 2 · Machine Learning Test & Score (k-NN, Árvore, Reg. Logística) + ROC e k-Means/PCA .ows
Arquivos .ows na pasta Orange/. Se o widget File pedir o caminho, aponte para os CSV da pasta Dados/.

Notebooks em Python

.ipynb · mesmo pipeline em código
  • 01 · Análise e processamento de dados Exploração, limpeza, transformação e amostragem — Dia 1 Colab .ipynb
  • 02 · Treinamento de modelos Classificação, avaliação (ROC, métricas) e clustering — Dia 2 Colab .ipynb
Abra direto no Google Colab (sem instalar nada) ou baixe o .ipynb para rodar no Jupyter/VS Code. Os dados estão em Dados/.

Agenda do módulo

O fio condutor é o caso Tramontina: do dado bruto à decisão de negócio. A teoria de cada bloco é seguida de prática no Orange.

Dia 1 — Data Mining6h
  • Módulo 1O que é Data MiningDefinições, KDD · CRISP-DM · SEMMA e o fluxo de um projeto
  • Módulo 2Dados e estatística descritivaTipos de atributos, medidas de centro e dispersão, distribuição, covariância e correlação
  • Módulo 3ArmazenamentoBancos relacionais (SQL), NoSQL, vetorial (embeddings/RAG) e formatos
  • Módulo 4Pré-processamentoLimpeza (ausentes, duplicatas, outliers), LGPD, transformações e redução de dimensionalidade
  • Módulo 5Amostragem e geração de dadosAmostragem estratificada, desbalanceamento, SMOTE e dados sintéticos
  • PráticaOrange · ExploraçãoPipeline completo de exploração e preparo da base
Dia 2 — Machine Learning4h
  • Módulo 1O que é Machine LearningDefinição, tipos de aprendizado, fluxo clássico, modelo × algoritmo e IA/ML/DL
  • Módulo 2Aprendizado supervisionadok-NN, árvores, regressões linear e logística, e ensembles
  • Módulo 3Avaliação e validaçãoTreino/validação/teste, validação cruzada, matriz de confusão, métricas e AUC
  • Módulo 4Não-supervisionadok-Means, escolha de k (cotovelo/silhueta) e regras de associação
  • Módulo 5Redes neurais e Deep LearningNeurônio artificial, redes profundas, CNNs e Transformers
  • PráticaOrange · ModelagemClassificação comparada e clustering
