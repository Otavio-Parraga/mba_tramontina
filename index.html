Workflows do Orange

.ows · pipelines prontos
  • Dia 1 · Data Mining Exploração: File → Data Table → Distributions, Scatter, Preprocess, Sampler .ows

Arquivos .ows na pasta Orange/. Se o widget File pedir o caminho, aponte para os CSV da pasta Dados/.

Notebooks em Python

.ipynb · mesmo pipeline em código

  • 02 · Treinamento de modelos Classificação, avaliação (ROC, métricas) e clustering — Dia 2 .ipynb
Notebooks Jupyter na pasta Notebooks/. Abra com Jupyter, VS Code ou Google Colab; os dados estão em Dados/.

Como abrir os workflows

  1. Instale o Orange Data Mining (gratuito) — veja o manual de instalação passo a passo para Windows e macOS.
  2. Baixe o conjunto de dados correspondente e o arquivo .ows.
  3. Abra o .ows no Orange. Se o widget File pedir o caminho, aponte para o CSV que você baixou.
